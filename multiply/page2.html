✖️ Let's Multiply by 3! 🧮

Welcome to the 3 Times Table!

Today we'll learn to count in groups of 3! 🚀

If you have 2 groups of cars and each group has 3 cars, you can say 2 × 3 = 6! That's much faster than adding 3 + 3! ✨

Let's explore the 3 times table step by step! 🎉

🚗 3 × 1 = 3

3 × 1 = 3

1 group of 3 cars = 3 cars total! 🚗

3 × 1 = 3
🚗 🚗 🚗
1 group of 3

🧠 Understanding 3 × 1

1 3 × 1 means 1 group of 3.
2 It's the same as just 3 — adding three once!
3 3 × 1 = 3 — any number × 1 is itself! 🎉

🚗 3 × 2 = 6

3 × 2 = 6

2 groups of 3 cars = 6 cars total! 🚗🚗

3 × 2 = 6
3 + 3 = 6
🚗 🚗 🚗
Group 1
🚗 🚗 🚗
Group 2

🧠 Understanding 3 × 2

1 3 × 2 means 2 groups of 3.
2 It's the same as 3 + 3 (adding 3 twice)!
3 3 + 3 = 6, so 3 × 2 = 6! 🎉

🚗 3 × 3 = 9

3 × 3 = 9

3 groups of 3 cars = 9 cars total! 🚗🚗🚗

3 × 3 = 9
3 + 3 + 3 = 9

🧠 Understanding 3 × 3

1 3 × 3 means 3 groups of 3.
2 It's the same as 3 + 3 + 3 (adding 3 three times)!
3 3 + 3 + 3 = 9, so 3 × 3 = 9! 🎉
