📐 Rate of Change with Shapes

Shapes grow proportionally!

When a shape grows, all its sides grow by the same ratio. This is another form of rate of change! 🚀

If a square is 2 × 2 and its height doubles to 4, the width also doubles to 4. The ratio stays the same! ✨

⬜

Growing a Square

A square has equal width and height. A 2 × 2 square has height = 2 and width = 2. If the height grows to 4, what is the width?

2 × 2
→
2 × 2
2

⬜ For a square, width always = height!   Ratio = 1 : 1

🔺

Growing a Triangle

This triangle has height = 2 and width = 3. The ratio is width : height = 3 : 2. If the height grows to 4, what is the width?

h=2, w=3
→
h=2, w=3
2

🔺 Width = height × 1.5  |  Ratio = 3 : 2

▬

Bonus: Rectangle with Same Ratio

A rectangle has height = 1 and width = 4 (ratio 4:1). If the height grows to 3, what is the width?

h=1, w=4
→
h=1, w=4
1

▬ Width = height × 4  |  Ratio = 4 : 1

🧠 Understanding Shape Rate of Change

1 Ratio tells us how width and height are related — it's always the same for a shape.
2 For a square, height = width (ratio 1:1). Double the height → double the width!
3 For the triangle, width = height × 1.5 (ratio 3:2). Height 2 → width 3. Height 4 → width 6!
4 For the rectangle, width = height × 4 (ratio 4:1). Height 1 → width 4. Height 3 → width 12!
