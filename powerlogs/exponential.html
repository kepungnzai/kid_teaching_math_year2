🚀 Exponents & Exponential Growth!

Discover the amazing world of powers and exponentials! See how numbers can grow super fast!

🔢 What is an Exponent?

An exponent tells you how many times to multiply a number by itself. It's like a shortcut for repeated multiplication!

xn
x = base, n = exponent
=
x × x × x × ... × x
n times!

For example, 2³ means 2 × 2 × 2 = 8. The base (2) is multiplied by itself 3 times (the exponent).

📈 POWER Base is the Variable!

In a power, the base is the variable and the exponent is fixed. This means the base changes but the exponent stays the same.

x2
x squared — base changes, exponent = 2
→
1²=1   2²=4   3²=9   4²=16   5²=25
Same exponent (2), different bases!

Think of it like this: The exponent is the machine that does the same thing to whatever number you put in!

  • 1 If x = 3 and the power is x², then 3² = 3 × 3 = 9
  • 2 If x = 5 and the power is x², then 5² = 5 × 5 = 25
  • 3 If x = 10 and the power is x², then 10² = 10 × 10 = 100

🎯 Key idea: In a power, the base (variable) changes, but the exponent (fixed number) stays the same!

📈 EXPONENTIAL Exponent is the Variable!

In an exponential, the base is fixed and the exponent is the variable. This means the base stays the same but the exponent changes.

2x
2 to the power of x — base = 2, exponent changes
→
2¹=2   2²=4   2³=8   2⁴=16   2⁵=32
Same base (2), different exponents!

Think of it like this: The base is the engine that keeps multiplying, and the exponent tells you how many times!

  • 1 If x = 1 and the exponential is 2ˣ, then 2¹ = 2
  • 2 If x = 3 and the exponential is 2ˣ, then 2³ = 2×2×2 = 8
  • 3 If x = 5 and the exponential is 2ˣ, then 2⁵ = 2×2×2×2×2 = 32

🎯 Key idea: In an exponential, the base (fixed number) stays the same, but the exponent (variable) changes!

⭐ The BIG Difference!

x² vs 2ˣ

x² (Power) — the base is the variable. The exponent (2) is fixed.
2ˣ (Exponential) — the exponent is the variable. The base (2) is fixed.

🌍 Where Do We See This?

🔵 Power (x²) in Real Life

📐

Area of a Square

If a square has side length x, its area is x². Double the side → 4× the area!

💡

Light Brightness

If you're twice as close to a light, it looks 4× brighter (inverse square law)!

🌍

Gravity

Gravity gets weaker with distance². Twice as far → ¼ the gravity!

🔴 Exponential (2ˣ) in Real Life

🦠

Bacteria Growth

One bacterium splits into 2, then 4, then 8... That's 2ˣ growth!

💰

Compound Interest

Money in a bank grows exponentially! Your savings double over time.

📱

Viral Videos

One share → 2 shares → 4 shares → 8 shares! That's exponential spreading!
