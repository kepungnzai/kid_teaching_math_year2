🍎 Let's Learn Division! 🧺

What is Division?

Division is like sharing equally! Imagine you have some yummy apples and some baskets. Division means putting the same number of apples into each basket. It's like being a fair apple-sorter! 🍎➡️🧺

2 ÷ 2 = 1

Each basket gets 1 apple!

🧠 Division Quiz Time!

1. If you have 6 apples and 3 baskets, how many apples go in each basket?

✨ 6 ÷ 3 = 2 apples in each basket! 🍎🍎

2. If you have 8 apples and 2 baskets, how many apples go in each basket?

✨ 8 ÷ 2 = 4 apples in each basket! 🍎🍎🍎🍎

3. If you have 12 apples and 3 baskets, how many apples go in each basket?

✨ 12 ÷ 3 = 4 apples in each basket! 🍎🍎🍎🍎

4. If you have 20 apples and 4 baskets, how many apples go in each basket?

✨ 20 ÷ 4 = 5 apples in each basket! 🍎🍎🍎🍎🍎 Great job! 🌟
