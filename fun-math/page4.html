🧠 The Secret: Set Up the BASE First!

1 Identify the BASE ratio: 35 L → 100 km
2 Find the scale factor: Target distance ÷ Base distance = 420 ÷ 100 = 4.2
3 Multiply base fuel by scale factor: 35 × 4.2 = 147 L
4 Multiply fuel by cost per litre: 147 × 191.0¢ = $280.77
💡 Key insight: Always find the BASE first, then scale up! 🎯
